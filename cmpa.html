--- title: CMPA layout: page hide: true ---

CMPA (Computer vision and Machine learning based Platform for Agrochemicals) is an android platform for scanning the label of an agrochemical product using OCR technique. For this task, we tried and tested different APIs like Tesseract, Microsoft Computer Vision API for accurately reading the text on the label of the product. Finally, we incorporated the Google Vision API to recognize the text.

This project idea was short-listed for the IUPAC Summit. Details: https://nextgen.iupac2019.be/2019/03/12/responsible-use-of-agrochemicals/
